## anchor_scales,  anchor_ratios,  anchor_strides

![1551772845112](img/1551772845112.png)

Please see these two .py files

```
mmdetection/mmdet/models/anchor_heads/anchor_head.py
mmdetection/mmdet/core/anchor/anchor_generator.py
```

As for why there are five anchor_stride, because fpn wants to downsample rpn again, so it is equivalent to have five-layer fpn network.

![1551773045411](img/1551773045411.png)

当anchor_base_sizes为None时，使用anchor_strides充当anchor_base_sizes,然后用fpn每一层的
map使用对应anchor_base_size，结合所有的anchor_scales和anchor_ratios生成框，进行前后景的分
类。
